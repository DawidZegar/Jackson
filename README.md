# Ćwiczenie teorytyczne
**1. Dlaczego JSON jest lepszy od zwykłego pliku .txt?**
> - JSON ma ustrukturyzowany format, dzięki czemu łatwo zapisać i odczytać dane obiektów (pola i wartości). Jest też czytelny dla ludzi i aplikacji.

**2. Co się stanie bez pustego konstruktora w klasie Student?**
> - Biblioteka Jackson nie będzie mogła utworzyć obiektu podczas deserializacji i pojawi się błąd.

**3. Do czego służy transient przy serializacji?**
> - transient oznacza, że dane pole nie będzie zapisywane podczas serializacji (zostanie pominięte).
